# Directed Study: Security on AI Agents

**Student:** Samuel Jerome  
**Institution:** University of North Texas  
**Focus:** Security in Large Language Models and Agentic AI Systems

---

## Overview

This repository documents research conducted as part of a Directed Study course exploring security threats, attack vectors, and defenses in the context of AI agents and Large Language Models (LLMs). Topics include prompt injection, adversarial attacks, data poisoning, backdoor attacks, federated learning security, and agentic AI threat mitigation.
